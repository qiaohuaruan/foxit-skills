## 💡 项目简介 <a id="readme"></a>

`foxit-skills` 是一个专为提高办公效率、实现**快速功能交付**而设计的低耦合、高内聚技能库。内部沉淀了针对各类主流办公应用（如 Excel、自动化脚本、文档处理、AI 工作流等）的高频核心技能与自动化片段。

无论是作为独立微服务运行，还是集成到现有的自动化系统（如视频生成、批量数据处理）中，都能实现开箱即用。

---

## ✨ 核心特性 <a id="features"></a>

* **⚡ 极速交付**：拒绝从零构建，标准化的输入输出接口，复制或引入即可直接上线运行。
* **🧩 模块化设计**：按办公应用维度划分，各模块间完全解耦，支持按需组合。
* **🔒 隐私与安全**：核心逻辑支持本地化/边缘侧部署，不依赖第三方黑盒，严格保障业务数据隐私。
* **🛠 易于扩展**：提供标准扩展模板，开发者可在几分钟内封装并交付一个全新的应用技能。
